A basic calculator application built using Android Studio. This project demonstrates fundamental concepts of Android development, including layouts, activities, and event handling.

✨ Features
Addition, Subtraction, Multiplication, and Division operations

Clean and simple user interface

Input validation to prevent crashes

Lightweight and beginner-friendly codebase

📂 Project Structure
MainActivity.java / MainActivity.kt → Handles calculator logic

activity_main.xml → Defines the layout (buttons, text fields)

AndroidManifest.xml → App configuration

build.gradle → Dependencies and SDK setup

🚀 Getting Started
Clone the repository:

bash
git clone https://github.com/your-username/simple-calculator.git
Open the project in Android Studio

Run the app on an emulator or physical device

🎯 Purpose
This project is ideal for beginners learning Android development. It covers the essentials of building an interactive app and serves as a foundation for more advanced projects.
